Install Instructions:
1. In the setup folder and run 
dependencies.bat
2. In the setup folder install
Python 3.6
	i) IMPORTANT: click add
	to PATH, when installing
3. Open PythonPDF.vi
4. Change paths to match your
system
5. Run the VI
