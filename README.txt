Install Instructions:
1. Run Python installer
    a) IMPORTANT: Select "Add Python 3.6.0 to PATH"
        i) LabVIEW will not be able to run the Python Module without this
2. Run dependencies.bat
    a) This installs all required Python libraries
        i) If needed, check to ensure libraries are install using the "pip list" command in the command prompt
3. Ensure Excel is installed
